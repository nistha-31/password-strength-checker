# password-strength-checker
Python-based password strength checker using Streamlit that checks whether a password can be classified as weak, medium, or strong.

## Features
- Length-based scoring (8+ and 12+ character thresholds)
- Uppercase, lowercase, numeric, and special character detection
- Detection of common weak patterns (e.g., "password", "1234")
- Dynamic strength classification (Weak, Medium, Strong)
- Real-time feedback and improvement suggestions

## Technologies Used
- Python
- Streamlit
- Regular Expressions (re module)
